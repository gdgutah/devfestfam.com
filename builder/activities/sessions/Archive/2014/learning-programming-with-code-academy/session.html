Code Academy provides free online training for people interested in learning to code. Their browser based, interactive platform, allows students to write code and immediately see it's output; this generates a unique and stimulating learning environment. Their tracks cover everything you need to get started with programming. This session will cover their JavaScript track, which includes topics like syntax, data structures, objects, and functions used in JavaScript development. After learning some JavaScript we will talk about how to get the most out of Code Academy and where to go after you complete their tutorials.
