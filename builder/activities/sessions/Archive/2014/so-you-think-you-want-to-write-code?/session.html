Coding can be a lot of fun! In this session we will talk about why coding might be something you will love. There are a lot of cool things you can do with code. We will look at examples and some of them might surprise you. Many resources are available to help you get a jumpstart on learning and most of them are free! We will look at some and provide a good list to take with you. When we are done you should know if you would like to try writing code and how to get started. Did I mention FUN?!
